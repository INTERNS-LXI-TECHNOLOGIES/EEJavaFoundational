Dictionary

requirement 1: Develop a dictionary web application that alphabetically lists English words and their meanings. Users should be able to search words for their meanings
     Words should be displayed with their meaning, usage, word type(verb, noun, etc), etc.
     
requirement 2:  Implement authentication and authorization. Admin can delete, update and edit words and their meanings but non-admin users can only view and search words and their meanings. (implement security using tomcat-users.xml initially, and then implement using a MySQL database)

requirement 3: Convert the application to a maven application
