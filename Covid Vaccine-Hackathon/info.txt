A user should be able to fill his details in the form to take part in the covid vaccination 

The details to be filled in the request form are:-
Name, Age,Address,phone number, Email-Id,Blood Type, 
 
The admin should approve or reject the request of each applicant and should be able to perform CRUD operation on the applicants.

User should get his status by entering the phone number or Name.

Requirements:

Usage of arrays, interface and packages is necessary.
Use mysql as the database
Use jsp and servlets for UI and access mysql database
Maven project will be the ideal choice
Configure users in database or users.xml file of tomcat
